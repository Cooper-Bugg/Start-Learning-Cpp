# Start Learning C++

A comprehensive learning resource for C++ organized into modules and projects.

## Repository Structure

This repository contains two main learning paths:

- Cpp-Foundations - Core C++ fundamentals and foundational concepts
- Cpp-Starter-Pack - Beginner-friendly projects and exercises

## Code Policy

All code in this repository is hand-typed. AI is only permitted to assist with documentation comments and README files.

## Getting Started

1. Start with the Cpp-Foundations folder to learn the basics
2. Move on to Cpp-Starter-Pack for practical projects
3. Each folder contains its own README with specific instructions

## Contributing

When contributing to this repository:
- Hand-type all code
- Use AI responsibly for documentation only
- Focus on learning and understanding
